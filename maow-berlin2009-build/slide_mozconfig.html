Die Einstellungen - mozconfig

  • Eine mozconfig-Datei enthält alle Einstellungen für einen Build-System-Durchlauf: 
    # mozconfig-testext
# objdir setzten, relativ zum Quellcode
mk_add_options MOZ_OBJDIR=../objdir-testext
# nur testext-Erweiterung bauen
ac_add_options --enable-application=extensions
ac_add_options --enable-extensions=testext
# Dieses Beispiel braucht keinen Compiler
ac_add_options --disable-compile-environment
  • Unsere Erweiterung liegt in mozilla-central/extensions/testext/
  • Befehle zum "Bauen": 
    cd mozilla-central
MOZCONFIG=../mozconfig-testext make -f client.mk configure
cd ../objdir-testext
make -C extensions
  • Das fertige XPI-Paket findet sich in objdir-testext/dist/xpi-stage/ und eine "vorinstallierte" Version in objdir-testext/dist/bin/extensions
