Bug Bounties: Lessions Learned

  • I did a personal bug bounty program in 2007, pledging 2700 USD for work on 7 SeaMonkey bugs
  • 4 top-paid items stay unfixed after half a year, 3 simpler items done
  • Nice to get some money to help doing needed work
  • No substantial speeding up of high-bounty tasks
  • Mainly experienced developers picking up tasks
  • Some feel uncomfortable with selective payment of volunteers
  • Money is nice, but not the driving force for getting stuff done
