Localization

  • Nightlies repackaged to localized builds (make repackage-* in app/locales)
  • L10n files taken from mozilla.org/l10n CVS
  • Which locales are included is determined by all-locales files
  • Opt-in for extensions like ChatZilla and venkman via all-locales
  • Extension L10n done via langpacks, dependent on the main extension
